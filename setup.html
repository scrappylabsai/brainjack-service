BrainJack Device Setup

Fill in the device details below to create a configuration QR code.

3

Scan from App

Open BrainJack on your phone and scan the QR code to add the device.

Install the Agent

# One-line install (Linux & macOS)
curl -fsSL https://raw.githubusercontent.com/scrappylabsai/brainjack-agent/main/install.sh | bash
# PowerShell (run as Administrator)
irm https://raw.githubusercontent.com/scrappylabsai/brainjack-agent/main/install.ps1 | iex
