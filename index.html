How it works

Three steps. Zero friction.

🎤

Speak

Talk naturally into the BrainJack app. On-device ASR — nothing leaves your network.

→
🧠

Think

AI interprets your intent, formats data, resolves macros, and queues keystrokes.

→
⌨

Type

Keystrokes inject into whatever window has focus. USB HID or network agent — your choice.

Why BrainJack

Built different.

Not another clipboard app. BrainJack is a universal HID bridge that works where nothing else can.

🔌

Universal USB HID

Appears as a standard keyboard. Works with BIOS, login screens, legacy DOS, anything that takes keyboard input since 1995.

🔐

HIPAA-Local by Default

All voice processing stays on your network. Zero cloud dependency. Token auth, TLS, audit logging built in.

💻

Every Platform

Windows (SendInput), macOS (osascript), Linux X11 (xdotool), Linux Wayland (ydotool). One agent, all OSes.

⚡

Zero Install on Target

With the ESP32 dongle, the target machine needs nothing. No drivers. No software. No admin rights. Just USB.

🎛

Command Sheets

One-tap macros for vim, tmux, emacs, VS Code, shell. Tool-specific palettes, color-coded, OS-aware.

🚀

Open Ecosystem

Compatible with ESP32-S3, Flipper Zero, any WebSocket client. Source-available under BSL. Build on it.

Get Started

Run everywhere.

Install the agent on your target machine in 30 seconds. Or flash the ESP32 dongle for zero-install deployment.

Quick Install
# Clone and install (any platform)
git clone https://github.com/scrappylabsai/brainjack-agent.git
cd brainjack-agent
# Linux/macOS:
./install.sh
# Windows (PowerShell):
powershell -ExecutionPolicy Bypass -File install.ps1
